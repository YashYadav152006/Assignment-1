# Assignment-1
# ⚡ Pikachu UI Landing Page

A modern and stylish **Pikachu themed landing page** built using pure **HTML & CSS**.  
This project focuses on learning:

- UI Layout Design
- Flexbox Positioning
- Absolute & Relative Positioning
- Hero Section Design
- Layering with `z-index`
- Responsive Visual Structure
- Modern Web Styling
